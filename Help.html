Amplitude To MIDI

Convert the amplitude of an audio signal into MIDI Control data in real-time.


Audio Input

The system default input device is used for input. You will need to set it before starting the program via System Preferences - Sound

If you have already started the program, just set the new device, then quit and re-open.


Sending MIDI to another program (e.g. Logic, Live etc)

To send MIDI data internally (e.g. to Logic, Live etc) you may need to enable the IAC Driver as follows:

- Open Audio MIDI Setup (/Applications/Utilities/Audio MIDI Setup)

- Open the MIDI window (Window Menu, 'Show MIDI Window')

- Double click the IAC Driver icon

- Enable the checkbox 'Device is online' (top right)


To check that things are working properly I recommend using MIDI Monitor http://www.snoize.com/MIDIMonitor/


Starting/Stopping

Use the large black OFF button to toggle on/off


Input and Output ranges

The audio input and MIDI output ranges can be set using the blue range sliders, or by entering values into the number boxes above and below the range sliders.


Audio Range

The audio range slider can be useful if you have a quiet signal, but still want a large MIDI output range, 


Audio Controls

Attack and Release control how quickly the attack and decay of the audio signal are followed.

Smoothing controls how long it takes for the MIDI output signal to change from one value to another.


Generally, if you have a signal with very fast attack and decays (e.g. percussion) you'll want low values for all. If you have a smoother signal (e.g. vocal, string or wind instruments) higher values (e.g. 1) will give a suitably smooth MIDI signal.


Attack - 60dB convergence time for following attacks

Release  - 60dB convergence time for following decays

Smoothing - 60dB lag time in seconds


Presets are provided for Percussion and Vocal signals, you can also save your own presets. If you use the same name as an existing preset, it will be overwritten.


MIDI Controls


MIDI Output Device selects which MIDI device to send the data to. 

To send to another application select IAC Driver (you may need to enable this, see above), or to send to hardware select your MIDI interface.


MIDI Channel - The MIDI channel to send on. 

Can be any value from 1 - 16. 

Unless you are sending MIDI control values from other programs/devices you can leave this on 1.


MIDI Control Value - The control value to use.

Can be any value from 0 - 127

Many of these values have an expected function, and some programs automatically assign and protect them e.g. Logic uses 7 to control the current track's volume fader.

A popup menu of useful values is provided.

If you want to control something not on the list (e.g. FX mix), use the first item in the menu (3 - Freely Assignable) or any of the undefined channels.
